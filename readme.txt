== Description ==

Displays the latest movie trailers featured on Apple.com via the RSS Feed.

Each trailer listed comes along with the movie poster, synopsis of the film and Facebook and Twitter share links.

> Please note: The RSS feed is controlled by Apple so you may find at times that they have not updated it in comparison to what is shown on http://trailers.apple.com

== Installation ==

1. Unzip the archive of the plugin.
2. Upload folder `latest-apple-movie-trailers` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Do I need to edit any files? =
No, just activate the plugin and it should work.

= How do I display the latest apple movie trailers? =
Add this shortcode to either a post or page. `[apple_trailers]`

= How do I change how many trailers to display? =
Go to 'Settings -> Latest Apple Movie Trailers' and select how many you want to show from the drop down menu.

= How do I change the poster size? =
Go to 'Settings -> Latest Apple Movie Trailers' and select which size of the poster you want showing.

= I selected Extra Large for poster size and now the descriptions don't fit? =
This is best used on a full-width page if you have one.
